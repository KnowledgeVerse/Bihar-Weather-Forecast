# Bihar Weather Forecast System

A comprehensive web application for generating weather forecasts for Bihar districts with bilingual support (Hindi and English).

## Features

- **District Selection**: Interactive dropdown with all 38 Bihar districts
- **Weather Phenomena**: Multiple weather conditions with checkboxes
- **Bilingual Support**: Hindi and English forecast generation
- **Interactive Map**: Leaflet-based map for district visualization
- **Export Options**: Text and PDF export functionality
- **Responsive Design**: Mobile-friendly interface
- **Keyboard Shortcuts**: Ctrl+G (Generate), Ctrl+C (Clear)

## Weather Phenomena Supported

1. मेघगर्जन/वज्रपात (Thunderstorm/Lightning)
2. तेज़ हवा (Gusty Wind 30-40 kmph)
3. लू (उष्ण लहर) (Heat Wave)
4. ओलावृष्टि (Hailstorm)
5. भारी वर्षा (Heavy Rainfall)
6. घना कोहरा (Dense Fog)
7. शीत दिवस (Cold Day)
8. गर्म रात्रि (Warm Night)

## Installation

1. Extract the project folder to your desired location
2. Place your shapefiles in the `data/Bihar_Districts_Shapefile/` folder
3. Open `index.html` in a web browser or use VS Code Live Server

## Usage

1. Select a district from the dropdown
2. Choose weather phenomena using checkboxes
3. Click "पूर्वानुमान तैयार करें" to generate forecast
4. Export results as text or PDF
